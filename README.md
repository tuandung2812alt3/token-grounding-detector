# Token Grounding Detector

**Beyond the Global Scores: Fine-Grained Token Grounding as a Robust Detector of LVLM Hallucinations**

Code and data coming soon.

## Overview

A lightweight framework for token-level hallucination detection in Large Vision-Language Models. We introduce two novel metrics — **Attention Dispersion Score (ADS)** and **Cross-modal Grounding Consistency (CGC)** — that capture patch-level structural signatures of hallucinated tokens.
